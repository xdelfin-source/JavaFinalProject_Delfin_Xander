Title: Chordify
problem statement: For beginner musicians/students who wish to learn simple music theory.
program features: 
- Major & Minor Generation: Capable of constructing both Major and Minor triads based on music theory intervals.
- Smart Input Normalization: Automatically handles flat inputs (e.g., converting "Db" to "C#") to ensure the correct chord is found.
- Input Validation: Checks user input against a database of valid musical notes and prevents the program from crashing due to typos.
- Interactive GUI: Uses Java Swing (JOptionPane) to provide a user-friendly pop-up interface rather than a command-line text stream.
- Continuous Session: Allows the user to generate multiple chords in a row without restarting the program.
how to run the program: To run the program, save the code as Chordgen.java, then open your terminal and execute javac Chordgen.java followed by java Chordgen.
